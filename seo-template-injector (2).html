AI & API — Google Gemini 1.5 Flash
Indsæt nøgle for live indhold — uden nøgle genereres dummy-tekst
Ikke sat
Hent gratis nøgle på aistudio.google.com → Get API key. Nøglen forlader ikke din browser.
Sådan aktiverer du live AI-indhold Browsere blokerer direkte API-kald fra lokale filer. Løsningen er den medfølgende proxy.js:

1. Åbn Terminal / Kommandoprompt i mappen med filerne
2. Skriv: node proxy.js og tryk Enter
3. Åbn dashboardet på: http://localhost:3456

Kræver Node.js (gratis på nodejs.org). Proxy og dashboard skal ligge i samme mappe.
Dummy Mode: Fungerer uden proxy og API-nøgle — genererer placeholder-tekst så du kan teste template og layout.
